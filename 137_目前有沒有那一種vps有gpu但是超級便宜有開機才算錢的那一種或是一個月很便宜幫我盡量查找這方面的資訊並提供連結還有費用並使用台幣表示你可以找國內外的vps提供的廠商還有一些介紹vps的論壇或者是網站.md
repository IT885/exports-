### 二、申請 Google Cloud T4 GPU 步驟

1. **註冊 Google Cloud 帳號**
   - 前往 Google Cloud 官方網站，點選「開始免費試用」。
   - 使用 Google 帳號登入，填寫個人/公司資料。
   - 綁定信用卡或金融卡（僅作驗證，首度註冊贈送 US$300 試用金）。[3][4][5][6]

2. **建立專案與設定計費**
   - 建立新專案，並將帳號與專案綁定至計費帳戶。
   - 設定預算與警示，避免超支。[6][7][8]

3. **建立 GPU 虛擬機（VM）**
   - 進入 Google Cloud Console，選擇「Compute Engine」→「VM 實例」→「建立實例」。
   - 選擇地區（建議 us-central1 或 asia-east1），機型選「N1」或「E2」系列。
   - 在「GPU」選項中加選「NVIDIA T4」。
   - 設定所需 vCPU、記憶體、磁碟空間。
   - 完成設定後啟動 VM，即可使用 T4 GPU 運算。

4. **安裝 OCRFlux 及相關環境**
   - 於 VM 內安裝 Linux（Ubuntu/Debian），安裝 CUDA 驅動、poppler-utils、必要字型與 OCRFlux。[1]
