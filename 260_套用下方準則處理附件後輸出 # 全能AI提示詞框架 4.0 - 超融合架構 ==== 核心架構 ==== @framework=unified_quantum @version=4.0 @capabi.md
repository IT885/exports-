# Google Cloud 承諾使用折扣 (CUD) 超值攻略

## 基本概念與核心知識

承諾使用折扣 (CUD) 是 Google Cloud 為長期用戶提供的重要成本節省機制。透過承諾在特定時間內使用 Google Cloud 服務，您可獲得顯著折扣，最高可達 70% 的費用減免。

**折扣機制本質**：
- 本質上是「長期租約」概念，您承諾長期使用，Google 提供優惠價格
- 承諾期限分為 1 年或 3 年選項
- 折扣幅度隨承諾時間增長而增加

## 兩大 CUD 類型比較

### 依資源計算的 CUD
- **適用情境**：確定需要特定類型虛擬機器，且用量穩定
- **承諾期限**：1年或3年
- **折扣幅度**：最高可達55-70%
- **限制條件**：綁定在特定區域、專案和資源類型
- **典型應用**：長期穩定運行的生產環境

### Compute 彈性 CUD
- **適用情境**：用量可能變動，需要彈性使用不同區域或專案資源
- **承諾期限**：1年(28%折扣)或3年(46%折扣)
- **優勢特點**：可跨專案、區域使用，適用於多種服務
- **適用服務**：Compute Engine、GKE 和 Cloud Run
- **典型應用**：開發和測試環境，負載不穩定的應用

## 資金有限情況下的策略選擇

### 方案一：從小額彈性承諾開始
- 對穩定基礎用量購買彈性 CUD
- 例如：每月 $100 最低支出購買 1 年期彈性承諾
- 實際支付：$72/月 ($100 的 72%)
- 優勢：保留彈性，同時獲取穩定折扣

### 方案二：結合 Spot VM 節省更多
- 對非關鍵任務使用 Spot VM
- 價格比標準價格低 60-91%
- 風險：可能被 Google 隨時終止
- 應用場景：批次處理、容錯任務、開發測試環境

### 方案三：VM 自動化開關管理
使用 Python 腳本控制 VM 運行時間：

**自動化方案類型**：
- Cloud Functions + 排程器：完全無伺服器解決方案
- 本地電腦執行 Python 腳本：適合有常開設備的情況
- 基於使用率的智能控制：最省錢，自動偵測閒置時間

**成本節省估算**：
- 不使用自動控制：24小時 × 30天 × $0.01 = $7.2/月
- 基本時間控制 (工作時間)：$1.98/月 (節省 73%)
- 智能使用率控制：$1.39/月 (節省 81%)

## 遠端開關機解決方案

### 無伺服器控制架構
- **核心元件**：Cloud Functions + Cloud Scheduler
- **優點**：完全自動化，無需人工干預，成本極低
- **實現**：HTTP 觸發的 Cloud Function 控制 VM

### 消息平台控制方案

- **優點**：隨時手動控制，操作便捷，無需專門介面
- **實現**：Telegram Bot 接收指令並透過 Cloud Functions 執行

### Web 應用控制面板

- **優點**：視覺化界面，操作直觀，支援狀態監控
- **實現**：簡單 HTML/JS 頁面，可部署至 Firebase Hosting

- **優勢**：查詢功能豐富，擴展性好

- 利用自動化腳本控制資源
- 監控使用情況避免超額費用

通過明智結合這些策略，您可以在極低預算下實現 Google Cloud 服務的穩定高效運行，最大化每一分錢的價值。

# Google Cloud 低成本解決方案的白話範例與應用場景

## 服務類比與實際應用範例

**Cloud Run 的實際比喻：**
Cloud Run 就像是一個智能電燈，有人走進房間時才會亮，沒人時就關掉省電。如果您的應用是 WordPress，就像是一個需要持續供電的電視機，您可以這樣配置：
- WordPress 容器運行在 Cloud Run 上（設定最小實例為 1）
- 資料庫使用 Cloud SQL 存儲文章和用戶數據
- 媒體文件存儲在 Cloud Storage 中[1]

**實際場景舉例：**
假設您有一個旅遊部落格，當您上傳巴黎鐵塔照片時：
- WordPress 程式碼運行在 Cloud Run 上
- 照片描述和標題存在 Cloud SQL 數據庫中
- 照片本身存儲在 Cloud Storage 上
- 訪客打開網頁時，Cloud Run 從數據庫讀取內容，並從 Storage 加載圖片[1]

## Spot VM 與承諾使用折扣組合

**白話比喻：**
Spot VM 就像是折扣機票，非常便宜但可能會被臨時取消；而 CUD（承諾使用折扣）就像是健身房年卡，預先付款獲得長期折扣。[1]

**實際案例：**
假設您要運行一個小型網站：
- 正常 e2-small 每月成本：$15 美元
- 應用 3 年 CUD 後：$7.5 美元（節省 50%）
- 再應用 Spot 折扣：$3 美元（再節省 60%）
- 總節省：約 80%[1]

**資料保存情況：**
當 Spot VM 被終止時：
- 永久磁碟資料（如作業系統、網站檔案、資料庫檔案）會完全保留
- 臨時磁碟或記憶體中的資料會丟失
- VM 重新啟動後，通常會分配新的 IP 地址[1]

def is_work_time():
    """判斷當前是否為工作時間"""
