**回答**:
**RRT** 在網路環境中通常指的是 **RTT（Round-Trip Time，往返時間）**，這是衡量網路延遲的重要指標 。當大量 RTT 異常時，表示網路出現延遲或連線問題，需要緊急應變處理 。[1][2][3]

## RTT 基本概念

RTT 是指資料封包從來源傳送到目標地點，再返回來源所需的總時間，通常以毫秒（ms）為單位測量 。RTT 包含了傳播延遲、傳輸延遲、處理延遲和排隊延遲等多個組成部分 。[2][3][1]
