根據您的需求，以下是試算表公式：

## 基本公式

```
=IF(A2<TODAY()-60, TRUE, FALSE)
```

## 其他變化版本

**簡化版本（直接返回布林值）：**
```
=A2<TODAY()-60
```

**返回文字說明：**
```
=IF(A2<TODAY()-60, "超過60天", "未超過60天")
```

**返回天數差：**
```
=TODAY()-A2
```

**檢查是否超過60天並顯示天數：**
```
=IF(A2<TODAY()-60, "超過"&(TODAY()-A2)&"天", "未超過60天")
```

## 公式說明

- `TODAY()`：取得今天的日期
- `TODAY()-60`：今天往前推60天的日期
- `A2<TODAY()-60`：判斷A2的日期是否小於60天前
- `IF()`：條件判斷函數，如果條件成立返回TRUE，否則返回FALSE

建議使用第一個基本公式，它會在A2的日期小於60天前時返回TRUE，否則返回FALSE。
