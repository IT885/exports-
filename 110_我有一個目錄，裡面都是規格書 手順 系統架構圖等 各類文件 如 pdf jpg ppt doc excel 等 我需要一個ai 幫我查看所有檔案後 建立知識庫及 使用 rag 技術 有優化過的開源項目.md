import os
import pathlib
from ragflow import RAGFlow

def auto_scan_directories(root_path):
    """自動掃描多個目錄下的所有支援格式文件"""
    supported_formats = ['.pdf', '.docx', '.pptx', '.xlsx', '.txt', '.md', '.jpg', '.png']
    files_found = []
    
    for root, dirs, files in os.walk(root_path):
        for file in files:
            if any(file.lower().endswith(fmt) for fmt in supported_formats):
                files_found.append(os.path.join(root, file))
    
    return files_found

**為什麼這樣配置？**
- **第一階段**：專注於快速分析和決策，輕量向量化足以支援預掃需求[1]
- **第二階段**：需要高精度檢索，完整向量處理確保查詢品質[2]
- **效率考量**：避免在預掃階段進行昂貴的完整向量化操作

**向量搜尋實作**：[5]
```python
from pymongo import MongoClient
from sentence_transformers import SentenceTransformer

class MongoRAG:
    def __init__(self):
        self.client = MongoClient("mongodb://localhost:27017")
        self.db = self.client.cisco_knowledge
        self.collection = self.db.documents
        self.encoder = SentenceTransformer('BAAI/bge-large-zh-v1.5')
    
    def semantic_search(self, query, k=5):
        query_embedding = self.encoder.encode([query])[0].tolist()
        
        pipeline = [
            {
                "$vectorSearch": {
                    "index": "vector_index",
                    "path": "content_embedding", 
                    "queryVector": query_embedding,
                    "numCandidates": 100,
                    "limit": k
                }
            },
            {
                "$project": {
                    "doc_id": 1,
                    "metadata": 1,
                    "structured_data": 1,
                    "score": {"$meta": "vectorSearchScore"}
                }
            }
        ]
        
        results = list(self.collection.aggregate(pipeline))
        return results
```

3. **建構MongoDB系統**：使用主系統產出的結構化數據

- **API框架**：FastAPI（便於系統間通信）
